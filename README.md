## Requirements

* PHP v5.3+

* [SimpleTest](http://www.simpletest.org/)
* [Mockery](https://github.com/padraic/mockery#mockery)

The [xdebug](http://xdebug.org) PHP extension isn't strictly required, but is probably a good idea to have anyway.
